# Ghz.Life BFF (Node.js)

API BFF em Node.js + MySQL, seguindo estrutura modular com DDD.

## Como rodar

```bash
npm install
npm run dev
```

## Variáveis de ambiente

Copie `.env.example` para `.env` e ajuste conforme necessário.

## Swagger

Acesse `http://localhost:3333/docs` para testar as rotas.
